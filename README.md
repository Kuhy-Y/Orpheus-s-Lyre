# udist-p5js

这是一个基于 p5.js 的交互式网页作品，包含音频播放和串口/Arduino 连接功能。

## 核心运行文件

- `index.html`
- `style.css`
- `sketch.js`
- `libraries/p5.min.js`
- `libraries/p5.sound.min.js`
- `1.mp3` 到 `7.mp3`

## 本地预览

如果电脑装了 Node.js，可以在项目目录运行：

```powershell
node local-server.js
```

然后打开：

```text
http://127.0.0.1:8000
```

也可以直接运行：

- `start-preview.bat`
- `start-preview.ps1`
